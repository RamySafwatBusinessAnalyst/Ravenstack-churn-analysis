# Ravenstack-churn-analysis
End-to-end SaaS churn analysis project covering SQL preparation, EDA, feature engineering, modeling assessment, Power BI dashboards, and retention
## Dataset Attribution

This project uses the RavenStack synthetic SaaS dataset (multi-table), created by River @ Rivalytics.
